# Кредитный калькулятор

C#, ASP.NET MVC, Bootstrap

[Видео](https://youtu.be/PlDy4bsiHlk)

### Страница 1
* Поле "Сумма"
* Поле "Срок"
* Поле "Ставка в год"
* Кнопка "Рассчитать"

Поля проверяются на корректность введенных значений.

### Страница 2
График платежей в виде таблицы.
* Номер платежа
* Дата платежа
* Размер платежа по телу
* Размер платежа по %
* Итого, переплаты по кредиту

Формула взята [отсюда.](http://damoney.ru/bank/38_formula_annuitet.php)

Все расчеты на стороне сервера. Валидация на стороне клиента и на стороне сервера.

### Дополнительно
Реализован расчет платежей для ставки в день, чтобы платеж был не каждый месяц, а каждые 15 дней или 10 дней. Добавлено поле "Шаг платежа в днях" на первой странице.
